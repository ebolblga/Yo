## Introduction

To answer this question, let's first open the [rules of Russian orthography](http://orthographia.ru/orfografia.php?sid=11) and read:

### The use of the letter **ё** can be consistent or selective.

Consistent use of the letter **ё** is mandatory in the following types of printed texts:

  - in texts with consistently marked stress signs;

  - in books addressed to younger children;

  - in educational texts for younger schoolchildren and for foreigners learning Russian.

#### Note 1.
Consistent use of **ё** is adopted for the illustrative part of these rules.

#### Note 2.
At the author's or editor's discretion, any book may be printed consistently with the letter **ё**.

#### Note 3.
In dictionaries, words with the letter **ё** are placed in the general alphabet with words containing the letter **е**, for example: еле, елейный, ёлка, еловый, елозить, ёлочка, ёлочный, ель; веселеть, веселить(ся), весёлость, весёлый, веселье.

In ordinary printed texts the letter **ё** is used selectively. It is recommended to use it in the following cases.
  1. To prevent incorrect identification of a word, e.g.: всё, нёбо, лётом, совершённый (as opposed to все, небо, летом, совершенный), including to indicate the place of stress in a word, e.g.: вёдро, узнаём (as opposed to ведро, узнаем).

  2. To indicate the correct pronunciation of a word — either rare, insufficiently well-known, or commonly mispronounced, e.g.: гёзы, сёрфинг, флёр, твёрже, щёлочка; including to indicate the correct stress, e.g.: побасёнка, приведённый, унесённый, осуждённый, новорождённый, филёр.

  3. In proper names — surnames, geographic names, e.g.: Конёнков, Неёлова, Катрин Денёв, Шрёдингер, Дежнёв, Кошелёв, Чебышёв, Вёшенская, Олёкма.

## Study
> Homographs are words that are written the same way but are pronounced and/or have different meanings depending on the stress. In Russian, homographs often arise due to the optional use of the letter «ё».

Source: [Homograph - Wikipedia](https://en.wikipedia.org/wiki/Homograph).

In Russian there are homographs that differ only by the presence or absence of the letter «ё». This can lead to a change in the meaning of a word. In such cases, the letter «ё» acts like a diacritical mark, indicating stress and therefore a difference in meaning.

The purpose of the further research was to identify possible homographs involving the letter «ё» where replacing the letter with «е», whether intentionally or accidentally, changes the meaning of the word — or even the meaning of the whole sentence.

Important: homographs and homonyms are different things.

> Homonyms — words and other language units that are the same in spelling and sound but different in meaning.

Source: [Homonym - Wikipedia](https://en.wikipedia.org/wiki/Homonym).

## Algorithm
In the previous iteration of the project there was a suggestion to sort the final table by word popularity. To make that possible for the algorithm that finds pairs, a database containing word occurrence frequencies was used: [Baksalyar/mc.hertzbeat.ru-Frequency-Dictionaries](https://github.com/Baksalyar/mc.hertzbeat.ru-Frequency-Dictionaries/) — **480,092** words.

Next a list was formed of words that occurred both with the letter «ё» and with the letter «е» and it was sorted by the sum of frequencies. This produced a table with **4,929** unique pairs.

Unfortunately, the table contained a huge number of pairs where the word with the letter «е» was simply a typo.

It was then necessary to filter this table so that only rows remained where both words exist. For this the list of Russian lemmas and their derived forms by [Andrey Usachev](https://www3.smo.uhi.ac.uk/oduibhin/russian/rwordlist.htm) was ideal; he in turn used materials from A. A. Zaliznyak's morphological dictionary — **86,839** lemmas and **1,558,781** derived word forms.

After filtering, the list contained only **198** words; the first 5 words are shown below, and the full version is in [`/output/result.md`](https://github.com/ebolblga/Yo/blob/master/output/result.md).

## Conclusion

With free (non-mandatory) use of the letter «ё» there is a small chance of changing the meaning of a sentence, so the orthography changes of 1956 that allow voluntary use of this letter are **objectively** incorrect.

Yes, they look similar in writing, but that is where their similarity ends. Phonetically they sound different, and a person may misinterpret the meaning of your text.

### Example sentences
You can use LLMs where LLMs are truly good — composing text. For this I used the model [ChatGPT-5](https://chatgpt.com/):

1.
«У нас **всё** в порядке.»<br />
«У нас **все** в порядке.»<br />
— with ё: «всё» = "everything" (the whole, all together) — "всё в порядке" = "everything (things/affairs) is in order."<br />
— with е: "все в порядке" = "everyone is fine" — a different meaning (whole ≠ people).

2.
«Чтобы рассмешить публику, актёр надел маску и в шутку изображал, будто он — **осёл**, привлекая внимание к своей нелепой походке.»<br />
«Чтобы рассмешить публику, актёр надел маску и в шутку изображал, будто он — **осел**, привлекая внимание к своей нелепой походке.»<br />
— with ё: осёл = the animal (donkey).<br />
— with е: осел (without ё) can be read as the past tense of «осесть» — "sat down/settled" (depending on context) — meaning: "he pretended that he sat down" — a completely different action.

3.
«При обследовании полости рта врач аккуратно осмотрел **нёбо**, отметив участки с изменённым цветом.»<br />
«При обследовании полости рта врач аккуратно осмотрел **небо**, отметив участки с изменённым цветом.»<br />
— with ё: in the first variant the word clearly refers to the palate (in the mouth).<br />
— with е: the intended meaning suddenly shifts to the celestial sky.

Another excellent example where without contextual semantic analysis it is impossible to determine whether the letter «ё» should be placed in the last word of the following sentence from A. Tolstoy's novel *Peter I*:

4.
«Вот, скажут, такого господина нам бог послал, при таком-то **передохнём**…».<br />
«Вот, скажут, такого господина нам бог послал, при таком-то **передохнем**…».<br />
— with ё: «передохнём» (from "отдохнём") — "we will all rest."<br />
— with е: «передохнем» (from "передохнуть") — "we will all die."

## Yoficators and de-yoficators
After finishing the project I discovered a rabbit hole in the form of yoficators...

> Yoficator — a program or a text-editor module that replaces the letter «е» with «ё» in Russian-language texts according to Russian language rules where it should be used.

> De-yoficator — a program or a text-editor module that replaces the letter «ё» with «е» in Russian-language texts according to Russian language rules where it should be used.

Source: [Yoficator - Wikipedia](https://en.wikipedia.org/wiki/Yoficator).

All yoficators have their own lists of disputed words where, without semantic context analysis, it is impossible to decide whether to correct «е» to «ё» — literally the same problem as this project. Even GitHub repository names are very similar. After studying the word lists from open-source repositories it is hard to say who has the most complete list. They differ almost everywhere and deeper analysis is required.

## Technical information
To run the script from the terminal:

```bash
# Install the UV package manager
curl -LsSf https://astral.sh/uv/install.sh | sh
# powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex" on Windows

# Create virtual environment and install dependencies
uv sync

# Activate virtual environment
. .venv/bin/activate
# .venv\Scripts\activate on Windows

# Install pre-commit hooks
uv run pre-commit install

# Run pre-commit
uv run pre-commit run --all-files

# Ruff
uv run ruff format
uv run ruff check --fix

# Run the script
uv run python -m src.yo_pairs --validate both

# Possible arguments:
# --output: path to save the result
# --validate {none,first,second,both}: whether to filter the list obtained after the first step
# --no-download: do not download word lists
```
